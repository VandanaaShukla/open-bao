# open-bao
# 🔐 OpenBao Hands-On Lab

A self-learning project exploring OpenBao — an open-source secrets 
management tool — to understand why .env files aren't enough for 
production secrets management.

## What's inside

| Folder | Contents |
|--------|----------|
| `setup/` | Local installation guide using Docker |
| `examples/kv-secrets/` | Storing, retrieving and rotating secrets |
| `policies/` | Sample HCL policy files |
| `notes/` | Personal learnings and observations |

## Core concepts explored
- Authentication methods
- Policy-based access control
- KV secrets engine
- Secret rotation

## How to use this repo
Start with `setup/install.md` to get OpenBao running locally,
then follow the examples in order.
